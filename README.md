## Python

因 CMD 没有部分扩展需要的功能, 部分主程序/插件功能需要 Python 支撑

系统安装 Python 后, 可在 程序目录\ ```data\config\py_path.wcfg``` 中设置 Python 的可执行程序(```python.exe```).

- 如 Python 安装时设置了 ```%PATH%``` , 可直接设置为 ```python``` 或 ```py``` .
- 推荐的 Python 版本: **3.10**+
